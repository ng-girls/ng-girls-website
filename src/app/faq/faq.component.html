Frequently Asked Questions

Applying and Participating - ngGirls

How can I apply?

Just fill out the application form in the specific event's website - click on "Apply for the workshop". Do you already have experience in front-end development? Apply as a mentor!

Where is ngGirls going to take place?

We have many events all around the world. Check the homepage for the upcoming workshops.

What is the schedule of the workshop day?

The workshop usually takes place during the day. It starts with an introduction to Angular, and then the work is independent until the closing session, with a lunch break. You will get the updated schedule by mail before the event.

What is the criteria for choosing applicants?

You can read about it in our source of inspiration - Django Girls blog post.

Do I need prior knowledge in programming?

In the workshop we assume you have basic knowledge in programming. Once you know how it works in one language - it's easy to manage with JavaScript. Basic knowledge in HTML and CSS is also required, but it's really easy to learn.

I have no knowledge in programming whatsoever. But I’m willing to learn. Can you recommend me on some resources?

There are many tutorials on the internet for Web development: HTML, CSS and JavaScript. You should get the grasp of how things work. Codecademy is a good place to start, since it provides a platform for coding while learning.

Getting Ready

Should I bring my own laptop?

Yes! But if you don't have one, make sure you let us know in advance. It is also important for us that you will take home everything you’ll write and create during workshop.

Do I need to have something installed on my laptop?

We will send the full pre requirements to the participants prior to the workshop. One of the mentors will help you with the installations before the workshop. You can start now by installing NodeJS, Git and an IDE of your choice, and open an account on Github.

What IDE do you recommend?

There are various IDEs you can install with different licenses. Webstorm is very popular, and you can get it with a free trial of a month. Visual Studio Code is another great option which you can get for free!

Mentoring

Can I be a mentor?

Yes, we would love to have your help! If you have some knowledge and experience in front-end development, please fill out the form "call for mentors".

Are mentors being paid?

No, mentors, as well as the organizers, volunteer for the workshop.
